# 手写数字识别 - 神经网络数学原理实践

基于 NumPy 从头实现的神经网络，完全按照《手写识别神经网络数学原理》PPT 中的数学公式构建。

## 项目概述

本项目实现了一个完整的手写数字识别系统，包含：

- **神经网络核心**: 使用 NumPy 从零实现，不依赖 PyTorch/TensorFlow
- **Web 交互界面**: Flask 后端 + 手写画板前端，支持实时识别
- **完整的数学流程**: 像素展开 → 归一化 → 前向传播 → 交叉熵损失 → 反向传播 → 梯度下降

## 神经网络结构

```
输入层 (784) → 隐藏层1 (128, ReLU) → 隐藏层2 (64, ReLU) → 输出层 (10, Softmax)
```

对应 PPT 中的网络架构：
- 输入层: 784 个神经元 (28×28 像素展开)
- 隐藏层1: 128 个神经元，ReLU 激活
- 隐藏层2: 64 个神经元，ReLU 激活
- 输出层: 10 个神经元，Softmax 激活 (对应数字 0~9)

## 使用方法

### 1. 快速启动演示 (无需训练)

如果只想查看界面效果，可以直接生成演示模型并启动 Web 服务：

```bash
# 生成演示模型
python generate_demo_model.py

# 启动 Web 服务
python app.py
```

访问 http://localhost:5000 即可看到手写画板界面。

> **注意**: 演示模型未经训练，识别结果是随机的。

### 2. 训练模型

#### 快速训练 (约 1-3 分钟)

使用 10,000 张训练图片，训练 5 个 epoch：

```bash
python train.py --mode quick
```

#### 完整训练 (约 10-20 分钟)

使用全部 60,000 张训练图片，训练 20 个 epoch：

```bash
python train.py --mode full
```

#### 自定义训练参数

```bash
python train.py --mode quick --epochs 10 --lr 0.05 --batch-size 128
```

### 3. 启动 Web 服务

训练完成后，启动 Web 服务：

```bash
python app.py
```

访问 http://localhost:5000，在画板上手写数字，点击"识别数字"即可看到预测结果。

## 核心数学原理 (对应 PPT)

### 1. 数据预处理

#### 1-2 像素展开
将 28×28 的像素矩阵按行展开为 784 维向量：
```
x = [x₁, x₂, ..., x₇₈₄]ᵀ
```

#### 1-3 归一化
将像素值从 0~255 缩放到 0~1：
```
x_norm = x / 255
```

#### 1-4 One-Hot 编码
将标签 0~9 转换为长度为 10 的 One-Hot 向量：
```
y = [0, 0, ..., 1, ..., 0]ᵀ  (真实类别位置为 1)
```

### 2. 前向传播

#### 2-1 输入 → 隐藏层1
```
z⁽¹⁾ = W⁽¹⁾x + b⁽¹⁾
```

#### 2-2 隐藏层1 激活 (ReLU)
```
a⁽¹⁾ = ReLU(z⁽¹⁾) = max(0, z⁽¹⁾)
```

#### 2-3 隐藏层1 → 隐藏层2
```
z⁽²⁾ = W⁽²⁾a⁽¹⁾ + b⁽²⁾
```

#### 2-4 隐藏层2 激活 (ReLU)
```
a⁽²⁾ = ReLU(z⁽²⁾) = max(0, z⁽²⁾)
```

#### 2-5 隐藏层2 → 输出层
```
z⁽³⁾ = W⁽³⁾a⁽²⁾ + b⁽³⁾
```

#### 2-6 输出层 (Softmax)
```
ŷᵢ = softmax(z⁽³⁾ᵢ) = e^{z⁽³⁾ᵢ} / Σₘ e^{z⁽³⁾ₘ}
```

### 3. 损失计算

#### 3-1 交叉熵损失
```
Loss = -Σᵢ yᵢ log(ŷᵢ)
```

由于 y 是 One-Hot 向量，简化为：
```
Loss = -log(ŷₖ)  (k 为真实类别)
```

### 4. 反向传播

#### 3-2 输出层误差
```
δ⁽³⁾ = ŷ - y
```

#### 3-3 输出层梯度
```
∂Loss/∂W⁽³⁾ = a⁽²⁾ · δ⁽³⁾ᵀ
∂Loss/∂b⁽³⁾ = δ⁽³⁾
```

#### 3-4 误差传回隐藏层2
```
δ⁽²⁾ = f'(z⁽²⁾) ⊙ (W⁽³⁾ᵀ · δ⁽³⁾)
```

#### 3-5 隐藏层2梯度
```
∂Loss/∂W⁽²⁾ = a⁽¹⁾ · δ⁽²⁾ᵀ
∂Loss/∂b⁽²⁾ = δ⁽²⁾
```

#### 3-6 误差传回隐藏层1
```
δ⁽¹⁾ = f'(z⁽¹⁾) ⊙ (W⁽²⁾ᵀ · δ⁽²⁾)
```

#### 3-7 隐藏层1梯度
```
∂Loss/∂W⁽¹⁾ = x · δ⁽¹⁾ᵀ
∂Loss/∂b⁽¹⁾ = δ⁽¹⁾
```

### 5. 参数更新 (梯度下降)

```
W_new = W_old - η · ∂Loss/∂W
b_new = b_old - η · ∂Loss/∂b
```

其中 η 为学习率。
