# 雷云云文档 - 原型功能说明

## 一、项目概述

**项目名称**：雷云云文档  
**类型**：在线文档协作平台原型  
**技术栈**：HTML + CSS + JavaScript（单页应用）

---

## 二、页面结构

### 1. 登录页面
- **功能**：用户登录入口
- **特性**：
  - 左右分栏布局（左侧登录表单，右侧装饰动画）
  - 支持账号密码登录和验证码登录
  - 二维码扫描登录
  - 登录错误提示和锁定机制
  - 注册功能

### 2. 主页
- **功能**：文档管理中心
- **布局**：
  - 顶部导航栏（搜索、新建、用户头像）
  - 左侧边栏（置顶文档、我的文档库）
  - 主内容区（最近访问表格、快捷操作）

### 3. 文档编辑页面
- **功能**：在线文档编辑
- **特性**：
  - 富文本编辑
  - 段落等级设置（H1-H4）
  - 字体颜色和背景颜色
  - 左侧目录导航
  - 评论、剪切、复制、删除

### 4. 管理后台
- **功能**：文件管理
- **特性**：
  - 文件列表展示
  - 状态筛选（正常、回收站、彻底删除）
  - 复选框批量操作
  - 搜索功能

---

## 三、核心功能模块

### 1. 用户系统
| 功能 | 说明 |
|------|------|
| 登录 | 支持密码登录和验证码登录 |
| 注册 | 新用户注册 |
| 用户下拉菜单 | 管理后台、主题设置、意见反馈、退出登录 |
| 退出登录确认 | 二次确认弹窗 |

### 2. 文档管理
| 功能 | 说明 |
|------|------|
| 新建文档 | 创建新文档 |
| 置顶文档 | 将常用文档置顶显示 |
| 最近访问 | 显示最近打开的文档 |
| 我的文档库 | 个人文档列表 |
| 删除文档 | 删除确认 + 回收站机制 |
| 搜索文档 | 按名称搜索 |

### 3. 文档编辑
| 功能 | 说明 |
|------|------|
| 段落等级 | T/H1/H2/H3/H4 |
| 字体颜色 | 6种预设颜色 |
| 背景颜色 | 5种预设颜色 + 透明 |
| 评论 | 添加评论 |
| 剪切/复制/删除 | 文本操作 |
| 目录导航 | 自动根据标题生成目录 |

### 4. 管理后台
| 功能 | 说明 |
|------|------|
| 文件列表 | 展示所有文件 |
| 状态筛选 | 按状态筛选文件 |
| 批量操作 | 复选框选择多个文件 |
| 文件图标 | 选中后显示文件图标 |

- 全局变量管理应用状态

## 七、使用说明

1. 打开 `prototype.html` 文件
2. 点击登录进入主页
3. 双击文档进入编辑页面
4. 点击用户头像进入管理后台
